Movie Night planning app

This app was created to make the process of choosing the movie easier. It uses YouTube and Movie DB APIs to retrieve information.

Begin by entering title of the movie in a search box. Trailers from YouTube for that movie will be displayed first. Select Movie DB to see more information about searched movie.

If you don't know the title of the movie, you can start by checking out Most Popular movies on Movie DB or Other links for more inspiration. Enjoy!
